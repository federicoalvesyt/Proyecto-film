🎬 NetFilm TV

NetFilm TV es una aplicación web tipo Netflix creada con HTML, CSS y JavaScript. Permite agregar, buscar y reproducir películas directamente desde el navegador.


---

🚀 Características

📺 Reproductor integrado (modal)

➕ Agregar películas con imagen personalizada

🔍 Buscador en tiempo real

💾 Guardado automático con LocalStorage

🗑️ Eliminar películas

🌐 Integración con Vexo TV dentro de la app



---

🛠️ Tecnologías usadas

HTML5

CSS3

JavaScript (Vanilla)



---

📦 Cómo usar

1. Abrí el archivo index.html


2. Agregá una película:

Título

Imagen (archivo desde tu dispositivo)

Link (YouTube u otro)



3. Tocá Agregar


4. Usá el buscador para encontrar contenido


5. Tocá Ver para reproducir




---

🎥 Integración Vexo TV

Vexo TV está integrado como una "película"

Se abre dentro del reproductor

No redirige fuera de la web



---

⚠️ Aviso legal

NetFilm TV no aloja contenido audiovisual. Solo reproduce enlaces proporcionados por el usuario.
